# Notes API

## Description
Mini project after CH01-03 from Node.js course.
File-based Notes API using http and fs modules.

## Tech Stack
- Node.js
- fs module
- http module
- nodemon

## Features
- Create, Read, Update, Delete notes
- File-based storage
- Simple REST-like API

## How to Run
1. npm install
2. npm run dev
3. Test endpoints using Postman
